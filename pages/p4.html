Harvard CS50W: Python and Web development

I completed the CS50W: Python and Web development, during my summer holidays of my senior year at high school. This course included learning about the design and implementation of web apps with Python, JavaScript, and SQL using frameworks like Django, React, and Bootstrap. It also included six hands on projects which tried to explore database design, scalability, security, and user experience etc. I really enjoyed the challenge while completing the project, I also learned a lot. I would really like to thank the CS50W staff for coming up with such brilliant infrastructure online.

These are the projects I made while attending the course: - Project 0, Project 1, Project 2 (wiki), Project 3 (search), Project 4 (mail), Project 5 (network), Project 6 (capstone- Paradox).
